# 债务管理重构 + 周期规则精简 + 真实响铃

## 1. 周期规则精简（移除自定义）
- `ReminderShopPage.tsx`：`PERIOD_PRESETS` 仅保留 **单次 / 每天 / 每周 / 每月 / 每季度 / 每年** 六种；删除自定义输入逻辑，改简单受控下拉 `PeriodSelect`。
- `shared/src/time.ts`：`computeNextFire` 增加 每天/每周/每月 解析，并保留 `每N周/每N月/每半年` 等旧值以兼容既有数据（向后兼容）。
- 冒烟验证：9/9 通过（6 预设 + 3 旧值）。

## 2. 提醒真实响铃（不再只是视觉提示）
- `web/src/lib/sound.ts`：用 **Web Audio API** 合成柔和双音提示音（C5→G5），无需任何音频文件；`unlockAudio()` 在首次用户手势解锁音频上下文（绕过浏览器自动播放限制）。
- `web/src/features/reminder/useReminderAlarm.ts`：监听所有钟，进入 `due/overdue`（真正到点）时播放；用 `rungRef` 防重复；处理后（完成/推迟/重置）清除以允许下次重响；单次变 `done` 不再触发。
- `App.tsx`：全局挂载 `useReminderAlarm()`，首次 `pointerdown` 解锁音频。

## 3. 债务管理表单重构（仅保留需求字段）
字段收窄为：**名称 / 类型（房贷·信用卡·消费贷·抵押贷·其他）/ 还款方式（等额本息·等额本金·先息后本）/ 本金 / 利率 / 期限 / 起始日期 / 每月扣款日 / 利率重定价日 / 利率重定价记录（多条）/ 提前还款记录（多条）/ 备注**。
- 移除 月最低、利率类型/基准利率/利率加点、高级 JSON 文本框。
- `利率重定价记录` 与 `提前还款记录` 改用**结构化「多条记录」编辑器**（增/删行），不再要求手填 JSON。
- 后端无需改动（字段早已支持）。

## 4. 还款清单（已还 / 待还）
- 点击债务条目（整行）即可展开。
- `DebtScheduleView` 查询该债务的 `debt_payment` 流水，按 `已还期数` 拆分计划表为 **「已还清单」（绿）** 与 **「待还清单」（琥珀）**，已还行标注实际还款日期；全部还清显示 🎉。
